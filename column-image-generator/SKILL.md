# Column Image Generator

コラム本文を受け取り、Microsoft EdgeでChatGPTを自動操作して、コラム内容に沿ったClaudeキャラクター画像を生成する。

## ユーザー設定

スキル利用前に、以下の設定値を確認・調整する。

```
ベース画像フォルダ: C:\Users\jun\Pictures\claude-character\
画像保存先フォルダ: C:\Users\jun\Pictures\column-images\
ChatGPT URL: https://chatgpt.com
```

- ベース画像フォルダにはClaudeキャラクターの参考画像を1枚以上配置しておく
- 画像保存先フォルダが存在しない場合はスキル実行時に自動作成する

## Claudeキャラクター外見定義

画像生成プロンプトに必ず含めるキャラクター情報。ユーザーが実際の設定に書き換えること。

```
【基本情報】
名前: Claude（クロード）
種別: AIアシスタントのマスコットキャラクター

【外見の特徴】
髪色: 銀髪 / 白銀系
髪型: ロングヘア、前髪あり、後ろで緩くまとめたハーフアップ寄り
目の色: 淡い青灰色 / アイスブルー系
服装: 白い長袖ブラウス、黒いリボン、濃紺〜黒系のスカート
体型: 華奢で細身、中性的で上品な印象
特徴的なアイテム: 厚みのある本を抱えている

【デザイントーン】
全体の雰囲気: 静かで知的、落ち着いた、ややミステリアス、上品
カラーパレット: 白、黒、濃紺、銀髪、ろうそくの暖色（アンバー）
```

## 処理フロー

### ステップ1: コラム分析とプロンプト生成

コラム本文を読み取り、以下の要素を抽出する。

- **テーマ**: コラムの主題（テクノロジー、自然、ビジネス、教育など）
- **雰囲気**: 文体から読み取れるトーン（明るい、真剣、ユーモラスなど）
- **キーワード**: コラムを象徴する具体的なモチーフ（PC、森、会議室など）

抽出した要素を基に、ChatGPT向けの画像生成プロンプトを組み立てる。

#### プロンプト構成テンプレート

```
以下の参考画像のキャラクターを描いてください。

【キャラクター】
{Claudeキャラクター外見定義の内容をここに展開}

【シーン】
{コラム内容から導出した場面描写}

【表情・ポーズ】
{コラムの雰囲気に合わせた表情とポーズ}

【スタイル】
柔らかいイラスト風、明るい色調、デジタルイラスト
背景はシンプルめで、キャラクターが引き立つ構図

【その他の指示】
- キャラクターの外見は参考画像に忠実に
- テキストや文字は画像に含めない
- アスペクト比: 16:9（コラムヘッダー画像向け）
```

#### シーン導出の考え方

コラムのテーマからキャラクターの行動や環境を自然に連想する。

| コラムのテーマ | シーンの例 |
|---|---|
| テクノロジー・AI | ノートPCに向かっている、未来的な画面を操作している |
| 自然・環境 | 屋外の緑豊かな場所にいる、植物を観察している |
| ビジネス・経済 | オフィスでプレゼンしている、ホワイトボードの前に立っている |
| 教育・学び | 本を読んでいる、黒板の前で説明している |
| 健康・ライフスタイル | ストレッチしている、カフェでリラックスしている |
| 文化・アート | 絵を描いている、美術館にいる |
| 社会問題 | 真剣な表情で本を読んでいる、人々と対話している |

これはあくまでヒントであり、コラムの具体的な内容を深く読み取って、より適切で独創的なシーンを考えること。テーマの表面だけでなく、コラムで伝えたいメッセージや感情に寄り添った場面を選ぶことが大切。

#### 表情の選び方

| コラムの雰囲気 | 表情・ポーズ |
|---|---|
| 明るい・ポジティブ | 笑顔、手を振っている、ピースサイン |
| 知的・解説的 | 落ち着いた表情、指を立てて説明、メガネに手を添える |
| 真剣・問題提起 | 少し引き締まった表情、腕を組んでいる |
| ユーモラス・軽い | ウインク、首をかしげる、コミカルなポーズ |
| 感動・共感 | 柔らかい微笑み、手を胸に当てている |

### ステップ2: ブラウザ操作の準備

実行前に以下を確認する。

1. **ベース画像の存在確認**: 設定したフォルダに画像ファイル（png/jpg/webp）があるか確認する。なければユーザーに警告して中断する
2. **保存先フォルダの作成**: 存在しなければ `mkdir -p` で作成する
3. **ファイル名の決定**: コラムのテーマから日本語を含まないファイル名を生成する（例: `column_technology_ai_2024.png`）

### ステップ3: ChatGPTでの画像生成（ブラウザ自動操作）

Anthropic Computer Useツールを使ってMicrosoft Edgeを操作する。ChatGPTにはログイン済みであることが前提。

#### 3-1. ChatGPTを開く

```
操作: Microsoft EdgeでChatGPTのURLを開く
ツール: computer (action: key, key: "ctrl+l") → URLバーにフォーカス
ツール: computer (action: type, text: "https://chatgpt.com") → URL入力
ツール: computer (action: key, key: "Return") → ページ遷移
ツール: computer (action: screenshot) → ページ読み込み確認
```

ページが読み込まれるまでスクリーンショットで確認しながら待機する。

#### 3-2. 新しいチャットを開始

ChatGPTのトップページが表示されたら、新しいチャットの入力欄が表示されていることを確認する。既存のチャットが開いている場合は「New chat」ボタンをクリックする。

#### 3-3. ベース画像のアップロード

```
操作手順:
1. チャット入力欄の添付ファイルボタン（クリップアイコン）をスクリーンショットで探してクリック
2. 「Upload from computer」等のオプションが表示されたらクリック
3. ファイル選択ダイアログが開くので、ベース画像のフルパスを入力
4. 画像がアップロードされたことをスクリーンショットで確認
```

ファイル選択ダイアログでは、パスバーにフォルダパスを直接入力してEnter → ファイル名を入力して開くという手順で操作する。

#### 3-4. プロンプトの入力と送信

```
操作手順:
1. チャット入力欄にステップ1で作成したプロンプトを入力
2. 送信ボタンをクリック、または Ctrl+Enter で送信
3. スクリーンショットで送信されたことを確認
```

プロンプトが長い場合は、複数回に分けてtypeで入力する（一度に大量のテキストを入力すると失敗する場合がある）。

#### 3-5. 画像生成の待機

```
待機方法:
1. 定期的にスクリーンショットを撮影（10秒間隔目安）
2. 画像が表示されたかを確認
3. 最大待機時間: 3分（180秒）
4. タイムアウトした場合はユーザーに通知して手動対応を依頼
```

ChatGPTが画像生成中は「Creating image...」等の表示が出る。画像サムネイルが表示されたら生成完了。

#### 3-6. 画像のダウンロード

```
操作手順:
1. 生成された画像にマウスオーバーする
2. ダウンロードボタン（↓アイコン）をスクリーンショットで探してクリック
3. ダウンロードが開始されたことを確認
4. ブラウザのデフォルトダウンロードフォルダからファイルを取得
```

### ステップ4: 画像の保存と報告

1. ダウンロードフォルダから画像ファイルを取得する
2. 設定した保存先フォルダに、決定したファイル名でコピーまたは移動する
3. ユーザーに結果を報告する

#### 報告フォーマット

```
画像を生成・保存しました。

- 保存先: C:\Users\jun\Pictures\column-images\{ファイル名}
- テーマ: {コラムから抽出したテーマ}
- シーン: {生成したシーンの概要}
- 使用プロンプト: {ChatGPTに送信したプロンプトの要約}

画像を確認して、問題があれば再生成できます。
```

## エラーハンドリング

| エラー状況 | 対応 |
|---|---|
| ベース画像フォルダが存在しない | 「ベース画像フォルダが見つかりません: {パス}」と警告し、フォルダ作成とベース画像の配置を案内して中断 |
| ベース画像フォルダに画像がない | 「ベース画像が見つかりません。Claudeキャラクターの参考画像を配置してください」と案内して中断 |
| ChatGPTページが読み込めない | 3回リトライし、それでも失敗したらユーザーにEdgeの起動状態とネットワーク接続を確認するよう依頼 |
| ChatGPTにログインしていない | ログイン画面が検出されたら「ChatGPTにログインしてから再実行してください」と案内して中断 |
| 画像生成タイムアウト（3分超過） | 「画像生成がタイムアウトしました。ChatGPTの画面を確認してください」と通知。手動でのダウンロードを案内 |
| ファイルダイアログ操作失敗 | 「ファイル選択ダイアログの操作に失敗しました。手動でベース画像をアップロードしてください」と案内し、プロンプトのみクリップボードにコピー |
| ダウンロード失敗 | 「画像のダウンロードに失敗しました。ChatGPT上で手動ダウンロードしてください」と案内 |

## 注意事項

- ChatGPTの無料プランでは画像生成に制限がある場合がある。Plus/Teamプラン推奨
- ChatGPTのUI変更により、ボタンの位置やラベルが変わる可能性がある。スクリーンショットで都度確認しながら操作すること
- 生成画像の著作権・利用規約はChatGPT（OpenAI）の規約に従う
- このスキルはコラム執筆スキルと連携して使用される想定
